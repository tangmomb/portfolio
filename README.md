# Portfolio

A one-page static portfolio website built with Next.js, Tailwind CSS, and Shadcn UI.

Features sections for websites, graphic design, videos, and photos.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build

```bash
npm run build
```

## Technologies Used

- Next.js
- TypeScript
- Tailwind CSS
- Shadcn UI
